# Census Income Classification & Segmentation

## Requirements

- Python 3.8+
- Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn lightgbm jupyter
```

## Project Structure

```
jpmc-ccb-risk-takehome/
├── data/
│   ├── census-bureau.data          # Raw dataset (199,523 records)
│   ├── census-bureau.columns       # Column name definitions
│   └── census_preprocessed.csv     # Generated by 01_eda.ipynb
├── figures/                        # Generated plots
├── 01_eda.ipynb                    # Data exploration & preprocessing
├── 02_classification.ipynb         # Income classification model
├── 03_segmentation.ipynb           # Customer segmentation model
├── report.pdf                      # Project report
└── README.md
```

## How to Run

Run notebooks in order — each depends on the previous:

```bash
# Option 1: Interactive (recommended)
jupyter lab

# Option 2: Headless execution
jupyter nbconvert --to notebook --execute --inplace 01_eda.ipynb
jupyter nbconvert --to notebook --execute --inplace 02_classification.ipynb
jupyter nbconvert --to notebook --execute --inplace 03_segmentation.ipynb
```

> **Important:** Run `01_eda.ipynb` first — it generates `census_preprocessed.csv`
> which is required by the other two notebooks.

## Notebooks

| Notebook | Description | Key Output |
|---|---|---|
| `01_eda.ipynb` | EDA, missing value analysis, preprocessing | `census_preprocessed.csv`, 8 figures |
| `02_classification.ipynb` | Logistic Regression + LightGBM classifier | ROC-AUC 0.94, threshold analysis |
| `03_segmentation.ipynb` | PCA + K-Means customer segmentation | 4 customer segments, business profiles |

## Key Results

- **Classification:** LightGBM achieves ROC-AUC = 0.9404. Recommended decision threshold: 0.25 (precision 0.39, recall 0.77) for broad marketing, or 0.30 (precision 0.60, recall 0.48) for precision targeting.
- **Segmentation:** 4 customer segments identified. Cluster 1 (44.8% of population) contains prime working-age adults with the highest income rate (12.4% earn >$50K).
